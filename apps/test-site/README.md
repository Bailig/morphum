# Test Website

This is a test website for Morphum to load up in an iframe and test out the functionalities in dev mode.

This project is generated with [Vite](https://vitejs.dev/).

## Getting Started

Prerequisites:

1. NodeJS v18
1. pnpm

To work on the project:

```
cd apps/web
```

To run the project:

```
pnpm dev
```

To build the project:

```
pnpm build
```
